How Your Data is Stored

Local Storage Only

All data in MyTradeMate is stored locally on your device:

1. FlutterSecureStorage (for API keys)

  • Your Binance API keys are encrypted and stored in your device's secure keychain
  • iOS: Stored in iOS Keychain (hardware-encrypted)
  • Android: Stored in Android Keystore (hardware-encrypted)
  • Never transmitted to our servers (we don't have servers!)

2. SharedPreferences (for app settings)

  • Theme preference (light/dark mode)
  • Quote currency preference (USDT/EUR/USDC)
  • Paper trading mode toggle
  • Risk disclaimer acceptance flag

Third-Party Services

MyTradeMate connects to external services to provide functionality:

1. Binance API

  • Purpose: Fetch market data, portfolio balances, and execute trades
  • Data Sent: Your API keys (encrypted in transit via HTTPS)
  • Data Received: Market prices, portfolio balances, order status
  • Privacy Policy: Binance Privacy Policy

2. Coinpaprika API

  • Purpose: Fetch cryptocurrency logos
  • Data Sent: Cryptocurrency symbol only (e.g., "BTC")
  • Data Received: Logo image URL
  • Privacy Policy: Coinpaprika Privacy

3. RevenueCat (Subscription Management)

  • Purpose: Process in-app subscriptions (Premium plan)
  • Data Sent: Anonymous user ID, subscription status, purchase receipts
  • Data Received: Subscription status, entitlements, expiration date
  • Data Stored: Purchase history (required by Apple/Google)
  • Privacy Policy: RevenueCat Privacy Policy

4. Apple App Store / Google Play Store

  • Purpose: Process payments for Premium subscriptions
  • Data Collected: Payment information, purchase history (handled by Apple/Google)
  • Privacy Policy: Apple Privacy | Google Privacy

Important: All API calls are made directly from your device to these services. We do not act as a middleman. We do not log or store any requests.
