Q: How do I get started with MyTradeMate?

A: Follow these steps:

  1. Download the app from App Store or Google Play
  2. Complete the onboarding and accept the risk disclaimer
  3. Create a Binance account (if you don't have one)
  4. Create Binance API keys
  5. Add your API keys in Settings ‚Üí API Configuration
  6. Start trading!
Q: Do I need a Binance account?

A: Yes, MyTradeMate connects to Binance API to fetch market data and execute trades.

Create a free account at: https://www.binance.com

Q: How do I create Binance API keys?

A:

  1. Log in to Binance
  2. Go to Profile ‚Üí API Management
  3. Click "Create API"
  4. Set permissions:
    • FREE mode: Enable "Read Info" only
    • PREMIUM mode: Enable "Enable Spot & Margin Trading"
    • Important: Do NOT enable "Enable Withdrawals"
  5. Copy API Key and Secret Key
  6. Add them to MyTradeMate in Settings

üìä Trading

Q: What order types are supported?

A: MyTradeMate supports 4 order types:

  • Market Order: Executes immediately at current price
  • Limit Order: Executes only at your specified price (or better)
  • Stop-Limit Order: Triggers a limit order when price hits stop price
  • Stop-Market Order: Triggers a market order when price hits stop price
Q: What is API Mode?

A: API Mode determines how MyTradeMate connects to Binance:

  • FREE Mode (Read-Only): View portfolio and AI predictions only. No trading.
  • PREMIUM Mode (Trading): Full trading capabilities with real orders.

How to switch: Go to Settings ‚Üí API Configuration ‚Üí Add API keys with appropriate permissions

Tip: Start with FREE mode to learn the app, then upgrade to PREMIUM when ready to trade.

üêõ Troubleshooting

Q: App won't connect to Binance

A: Try these solutions:

  1. Check your internet connection
  2. Verify API keys are correct (no extra spaces)
  3. Check if keys are still active in Binance
  4. Verify API permissions are enabled
  5. Restart the app
  6. Check Binance status: Binance Status
Q: Orders not executing

A: Check these:

  • FREE mode (READ ONLY) cannot place orders - need TRADING permission
  • Check if you have enough funds
  • Binance has minimum order sizes (~$10-20 USD)
  • Verify your API key has "Enable Spot & Margin Trading" permission
